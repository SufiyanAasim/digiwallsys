# Support

- Use GitHub Issues for reproducible defects and feature proposals.
- Use GitHub Discussions for setup and usage questions when enabled.
- Include OS, Node version, Expo version, logs with secrets removed, and clear
  reproduction steps.
- Follow `SECURITY.md` for private vulnerability reporting.

Community support is best-effort. Never post `.env` contents, JWTs, database
URLs, passwords, or real transaction data.
