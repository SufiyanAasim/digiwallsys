## 📖 Overview

**Meridian** aligns wallet, ledger, and provider records with precise accounting.

## 🎯 Objectives

- Reconcile cached wallet balances against immutable ledger entries.
- Detect and record discrepancies without rewriting history.
- Enforce currency-specific precision and rounding rules.
- Produce settlement and exception reports.

## ✨ Planned scope

### 🧮 Reconciliation

- On-demand audited reconciliation runs.
- Discrepancy count, details, operator, and timestamp retention.
- Provider settlement and exception workflow foundations.
- Currency precision validation.

## 🏛️ Architecture progress

Reconciliation reads the append-only ledger as the financial reference and treats
cached wallet balances as independently verifiable projections.

## 💻 Compatibility

PostgreSQL 14+ · provider settlement exports
