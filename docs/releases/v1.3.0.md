## 📖 Overview

**Harbor** provides a protected operations dashboard for managing and reviewing
the wallet ecosystem.

## 🎯 Objectives

- Restrict administrator functions by current database role.
- Expose operational balances, funding, schedules, and risk counts.
- Review fraud events without changing financial history.
- Record every administrative action in the audit trail.

## ✨ Planned scope

### 🧭 Audited administration

- Administrator overview and exception metrics.
- Fraud review and disposition actions.
- Audit-log browser and reconciliation trigger.
- Approval-oriented operational controls.

## 🏛️ Architecture progress

Administrator authorization is checked against PostgreSQL on each protected
request rather than trusting a potentially stale token role.

## 💻 Compatibility

Expo SDK 53 · Node.js 20+ · PostgreSQL 14+
