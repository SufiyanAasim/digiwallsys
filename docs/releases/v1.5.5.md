**Status:** Production Release

## 📖 Overview

**Armada** is the production release milestone for the complete digiwallsys ecosystem.
It integrates managed cloud PostgreSQL on Supabase, production API hosting on Render,
and Web Dashboard deployment on Vercel, alongside dark mode UI styling and system credits.

## 🎯 Objectives

- Deploy managed PostgreSQL schema, triggers, and migrations to cloud database.
- Deploy Express API server to production containerized hosting.
- Export and host production Web Dashboard bundle with cross-platform session storage.
- Integrate sleek Dark Mode design system and system credits screen.
- Verify security, rate limits, CORS, and double-entry ledger stability end-to-end.

## ✨ Production Scope & Enhancements

### 🚢 Ecosystem & Cloud Deployment

- **Database**: Managed Supabase PostgreSQL with pooler connectivity and versioned migrations applied.
- **Backend API**: Render.com Web Service (`https://digiwallsys-api.onrender.com`) with automated health monitoring.
- **Web Dashboard**: Vercel production hosting with Expo Web Metro bundling and cross-platform session persistence.
- **UI & Experience**: Native Dark Mode theme, inline error alert banners, and dedicated Credits Screen.

## 🏛️ Architecture Progress

Armada connects all platform layers into a live, cloud-hosted environment. Every wallet
transaction remains balanced, idempotent, and auditable across Web, Android, and iOS clients.

## 💻 Compatibility

Supported Android/iOS releases · Node.js 20 LTS · Supabase PostgreSQL 16 · Render API · Vercel CDN
